# Iris-Flower-Classification-using-Machine-Learning
This project uses machine learning to classify Iris flowers into three species:

Setosa
Versicolor
Virginica

based on their physical measurements.

# Algorithm Used

K-Nearest Neighbors (KNN)

Dataset Features
Sepal length
Sepal width
Petal length
Petal width
# Workflow
Load dataset using Pandas
Split data into training and testing sets
Train KNN model
Predict test data
Evaluate accuracy
# Result

Model achieved high accuracy on test data (approx 95–100%).
